# 投研工作台 · Research Desk

个人股票投研工作台 — 移动端优先的单页 Web 应用。

## 功能

- **⚡ 高压信号流**：瀑布流展示自选股相关资讯，严守 76 只自选股能力圈
- **📊 自选全景**：分组 Tab（关注 / 沪深 / 港股 / 美股 / 翻倍），实时价格 tick + 迷你走势图
- **投研详情页**：滑入式详情，含价格面板、市场帖子、AI 深度调研工作区
- **AI 研判闭环**：输入推演逻辑 → 召唤 AI 定向检索 → 客观数据回填

## 运行

无构建工具，直接打开 `index.html` 或通过任意 HTTP 服务器访问：

```bash
python3 -m http.server 8765
# 访问 http://localhost:8765
```

## 数据

当前使用确定性种子随机数生成 mock 价格数据（按 symbol 哈希，刷新稳定）。
后续可替换为真实行情 API。

## 技术栈

- 纯 HTML / CSS / JS，无依赖
- Inter + JetBrains Mono 字体
- Navy/Slate 深色主题，语义色区分市场
